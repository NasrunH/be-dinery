Introduction

Selamat datang di dokumentasi API Dinery. API ini dirancang untuk aplikasi manajemen restoran couple (pasangan).

Base URL: http://localhost:5000/api
Auth Header: Authorization: Bearer <TOKEN>

Authentication

Endpoint untuk pendaftaran dan login user.

POST /auth/register
Register User
Body 
{
  "email": "user@example.com",
  "password": "password123"
}
POST /auth/login
Login
Response 
{
  "message": "Login Berhasil!",
  "token": "eyJhbGciOiVIUz...", // <-- Simpan ini!
  "user": { "id": "uuid...", "email": "..." }
}
GET /auth/me
Get My Profile

Mendapatkan data user yang sedang login.

PUT /auth/profile
Update Profile
Body 
{
  "display_name": "Rangga",
  "birth_date": "1995-10-20",
  "phone": "08123456789",
  "avatar_url": "https://..." // Dari endpoint Storage
}

Couple Management

Mengatur hubungan pasangan.

GET /couples/my-status
Check Status

Cek apakah user sudah punya pasangan atau belum.

POST /couples/create
Create Couple

User membuat couple baru. Mendapatkan invite_code.

{
  "name": "Rangga & Cinta"
}
POST /couples/join
Join Couple

User bergabung menggunakan kode invite.

{
  "invite_code": "XY78Z"
}
PUT /couples
Update Name 
{
  "name": "Keluarga Cemara"
}

Wishlist / Places

Manajemen tempat makan yang ingin dikunjungi.

POST /places/preview
Scraping Preview

Mengambil metadata (Title & Image) dari link TikTok/IG/Maps.

{ "url": "https://..." }
POST /places
Add to Wishlist

Menyimpan tempat baru.

{
  "original_link": "https://tiktok...",
  "platform": "TikTok",
  "meta_title": "Bakso Viral",
  "meta_image": "https://...",
  "maps_link": "https://maps.app.goo.gl/...",
  "name": "Bakso Pak Kumis",
  "category_id": 1,
  "price_range_id": 2,
  "tag_ids": [1, 3] // Opsional
}
GET /places
Get Wishlist

List semua tempat yang belum dikunjungi.

GET /places/nearby
Radius Search

Mencari tempat terdekat.

Query Params 
?lat=-7.8
&long=110.4
&radius=5  // (Dalam KM)
GET /places/gacha
Gacha (Random)

Memilih 1 tempat random dari wishlist.

PUT /places/:id
Update Place 
{
  "name": "Nama Baru",
  "tag_ids": [2] // Mengganti tag lama
}
DELETE /places/:id
Delete Place

Journal / Visits

POST /visits

GET /visits/history

Notifications

Master Data & Storage

GET /master/categories
Get Categories

POST /master/tags

Upload Image

Admin Dashboard

GET /admin/users/:id

DELETE /admin/users/:id

POST /admin/categories

POST /admin/tags
